# Online Course Management System - New Features Implementation

## Overview
This document outlines the new features added to the online course management system:
1. **Admin Course Reporting** - Generate comprehensive reports on courses
2. **Certificate Management Workflow** - Complete certificate lifecycle with teacher approval
3. **Student Certificate Access** - View and download certificates

---

## 2. CERTIFICATE SERVICE ENHANCEMENTS

### New Methods in `CertificateService.java`

#### Certificate Generation
- **`generateCertificateForCompletion()`** - Auto-generate certificate when course assignments are done
  - Takes: studentId, courseId, teacherId, finalScore
  - Returns: Certificate with PENDING approval status

- **`issueCertificate()`** - Manual certificate issuance (legacy method retained)

#### Certificate Approval Workflow
- **`getPendingCertificatesForTeacher(teacherId)`** - Get all pending certificates for teacher to review
- **`getAllPendingCertificates()`** - Get all system-wide pending certificates
- **`approveCertificate(certificateId, teacherId, comments)`** - Teacher approves certificate
  - Sets approvalStatus to "APPROVED"
  - Records teacher ID and approval date
- **`rejectCertificate(certificateId, teacherId, reason)`** - Teacher rejects certificate
  - Sets approvalStatus to "REJECTED"
  - Stores rejection reason

#### Certificate Distribution
- **`sendCertificateToStudent(certificateId)`** - Send approved certificate to student
  - Changes status from APPROVED to ISSUED
  - Records sent date
- **`getIssuedCertificates(studentId)`** - Get all issued certificates for a student
- **`getApprovedCertificatesForSending()`** - Get certificates ready to send

#### Certificate Formatting (Decorators)
- **`generateCertificateWithSignature(certificateId, signedBy)`** - Format certificate with teacher signature
- **`generateCertificateWithBadge(certificateId)`** - Generate achievement badge for high scores (≥80)

## WORKFLOW EXAMPLES

### Certificate Lifecycle Workflow

```
1. Course Completion
   ↓
2. Teacher Creates/Generates Certificate
   - Status: PENDING, ApprovalStatus: PENDING
   ↓
3. Teacher Reviews & Approves
   - GET /api/teacher/certificates/pending/{teacherId}
   - POST /api/teacher/certificate/{certificateId}/approve
   - Status: PENDING, ApprovalStatus: APPROVED
   ↓
4. Teacher Sends to Student
   - POST /api/teacher/certificate/{certificateId}/send
   - Status: ISSUED, SentDate: [current date]
   ↓
5. Student Downloads Certificate
   - GET /api/student/certificate/{certificateId}/download
   - Receives formatted certificate and badge
```

### Course Report Generation Workflow

```
1. Admin Requests Report
   - POST /api/admin/report/course/{courseId}
   ↓
2. System Gathers Data
   - Enrollments from EnrollmentRepository
   - Grades from GradeRepository
   - Certificates from CertificateRepository
   ↓
3. Report Generated
   - Stored in ReportRepository
   - Statistics calculated and formatted
   ↓
4. Admin Views Report
   - GET /api/admin/reports/{adminId}
```
